## 一、创建分支

```shell
git branch bugFix_01
```


## 二、切换分支

```shell
git checkout bugFix_01
```


### 创建并切换分支

```shell
git checkout -b bugFix_01
```


## 三、合并分支

```shell
git merge
```


## 四、删除分支

```shell
git branch -d bugFix_01
```


## 四、查看分支

```shell
git branch
```
