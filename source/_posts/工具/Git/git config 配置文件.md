## 一、配置文件分三个级别

- 仓库级别
- 用户级别
- 系统级别


## 二、如何编辑 `-e`

- `git config --local -e`
- `git config --glocal -e`
- `git config --system -e`


## 三、添加email与name

- git config --global user.email "email address"
- git cofnig --global user.name "name"
