## 三、Type 类型

- feat: 增加新功能；
- fix: 修复错误；
- docs: 修改文档；
- style: 修改样式；
- refactor: 代码重构；
- test: 增加测试模块，不涉及生产环境的代码；
- chore: 更新核心模块，包配置文件，不涉及生产环境的代码；
