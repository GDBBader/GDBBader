## 一、`git push`

push当前本地分支至当前分支所对应的远端分支


## 二、`git push origin master`

push 到origin仓库的master分支.
origin指向远端仓库.
可以使用`git remote -v`查看指向哪里.


## 三、git 里的`origin`是啥意思?

指的是远端的**仓库**别名.
