## 一、什么是npm

> NPM是随同NodeJS一起安装的包管理工具


可以用来管理

- 包
- 命令行程序

## 二、如何管理包

- 安装`npm install express`
- 使用
在代码中只需要通过 require('express') 的方式就好，无需指定第三方包路径。
`var express = require('express');`
